# TODO API
## Introduction
- This is a simple crud api for todo app
## Technologies
- Flask
- Flask-restx
- doker
- postgres

## End Points
- ### GET api/todo/get-data
- ### POST api/todo/add
- ### POST api/todo/get-data-name
- ### PUT api/todo/update
- ### DELETE api/todo/delete

## Running App
- docker-compose up --build
- docker exec -it <container_name> sh
